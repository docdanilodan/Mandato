# FinancePlus Mandato — Streamlit

Applicazione web compatibile con Streamlit Community Cloud.

## Avvio locale

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud

- Repository: cartella completa del progetto
- Main file path: `app.py`
- Python: 3.11 o 3.12 consigliato

Il programma non usa Tkinter e può quindi essere eseguito su un server Linux senza interfaccia grafica.
